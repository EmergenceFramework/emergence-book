Creating a Dynamic Page

Creating a Dynamic Page

<?php
print("Hello world!");

Why the <?php? So you can do this:

<html>
  <body>
    <p>
      <?php
        print("Hello, it&rsquo;s " . date('Y-m-d'));
      ?>
    </p>
  </body>
</html>

There are syntax shortcuts available for something so simple though:

<html>
  <body>
    <p>Hello, it&rsquo;s <?=date('Y-m-d')?></p>
  </body>
</html>
